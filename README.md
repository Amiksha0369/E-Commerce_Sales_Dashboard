# ## E-Commerce_Sales_Dashboard ##
📊 Power BI Data Visualization Project
* ** Overview**
This project focuses on creating interactive data visualizations using Power BI.
 The goal is to analyze the dataset and present insights through charts and tables.
 # ## Objective ##
To create meaningful visualizations from the dataset
To understand data distribution and patterns
To present insights in an easy-to-understand format
* ** Tools Used**
Power BI
Dataset (CSV file)
📂 Dataset Description
The dataset contains information such as categories, values (like sales/price), and other related fields used for visualization.
📊 Visualizations Created
1️⃣ Bar Chart
Used to compare values across different categories
Example: Sales by Category
2️⃣ Pie Chart
Used to show percentage distribution
Example: Category-wise contribution
3️⃣ Table (Optional)
Displays raw data in structured format
# ## Steps Performed  ###
Loaded dataset into Power BI
Selected Bar Chart from Visualizations panel
Added:
Category → X-axis
Values → Y-axis
Created Pie Chart using:
Legend → Category
Values → Numeric column
Customized chart titles and layout
📈 Key Insights
Identified which category has highest values
Observed distribution of data using pie chart
Compared different categories easily using bar chart
✅ Conclusion
Power BI helps in creating interactive and easy-to-understand visualizations.
This project demonstrates how raw data can be converted into meaningful insights using charts.
